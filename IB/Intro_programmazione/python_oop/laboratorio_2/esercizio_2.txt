Si vuole realizzare un programma per la gestione di una piccola banca. 

Il sistema deve prevedere le seguenti caratteristiche. 

1) Una classe che rappresenti il cliente
2) Una classe che rappresenti la banca 
3) Una classe che rapprsenti il conto corrente

Il sistema deve avere le seguenti caratteristiche: 

- Possibilità di creare un conto
- Versamento / prelievo 
- Stampa del resoconto bancario 
- Possibilità di creare un conto "Special" che differisce da quello base per la possiblità di 
creare un conto con un limite di prelievo giornaliero.
